# HTML
## HTML 구조태그
* HTML은 **웹문서**를 뜻한다.
*<meta name="description" content="">
<meta name="keywords" content=""> 까지 플러스 작성해야 함
* `태그` 웹문서에 들어가는 내용~제목 모든 요소는 `<태그>`로 묶여서 만들어진다.
## HTML 주의사항
* 파일명.html로 입력해야함(공백주면 안됨)
* 모든 파일명, 폴더명은 기본이 영문 소문자 사용,대문자 사용하는 경우도 있음, 특수문자 금지, 공백 금지, 한글 금지, 숫자 앞 금지
* start.html = o
* start1.html = o
* 1start.html = x (숫자앞)
* basic start.html = x (공백)
* basic_start.html = o (허용 특수문자 _ -)
* basic%start.html = x (허용 외 특수문자)
* 시작.html = x (한글)

## 태그속성 class, id
### class
* 모든 파일명, 폴더명은 기본이 영문 소문자 사용,대문자 사용하는 경우도 있음, 특수문자 금지, 공백 금지, 한글 금지, 숫자 앞 금지
* `<태그 class="값"></태그>`
* 2번 이상 반복하는 이름인 경우
* 예) lnb, snb, container 등
### id
* `<태그 id="값"></태그>`
* 반복하면 안되는 단독 명칭인 경우
* 예) header, gnb, footer 등
### 태그하나당 속성명이 중복될 순 없다.
### 내용
* `p` : 블록태그, 한줄~여러줄 단락형태의 내용일 때
* `br` : 인라인태그, 줄바꿈할때 사용
* `em` : 인라인태그, 단순 문맥 강조, 주로 p 자식으로 사용
* `strong` : 인라인태그, 중요+심각성 강조, p 자식 위주
* `blockquoto` : 블록태그, 긴 인용문
* `q` : 인라인태그, 짧은 인용문
* `code` : 인라인태그, 컴퓨터 명령어를 화면에 표시할때
* `address` : 블록태그, footer 안 주소 및 연락처, 해당 서비스 주소만 포함,address 안 블록 사용 주의(유효성에러)
### 레이아웃
* `div` : 블록태그, 그룹, 자식이 2개 이상이며 같은 방향일때, 블록과 인라인을 자식으로 모두 가질 수 있음
* `span` : 인라인태그, 그룹(위 div와 같음), 보조(디자인), 인라인만 자식으로 가질 수 있음.
* `hr` : 블록태그, 경계구분용(디자인 테두리 용도 아님)
### 특수문자 태그
* `&copy;` : footer 저작권마크 ⓒ
## 링크태그
* 필수구조 : ` <a href = "이동주소">클릭대상></a>`
* 링크 새창 열기 속성 : ` <a target=" blank>클릭대상</a>
###링크태그 href 주소 작성법
* 절대경로 : `href="https://주소"` 실제 서비스되는 웹주소
* 상대경로 : `href="./폴더 및 파일"` 같은 위치 또는 같은 위치
* 상대경로 : `href="../폴더 및 파일"` 상위 위치
### 바로가기링크(스크롤 이동하며 특정 위치로 가기)
1. `<a></a>` 태그 준비
2. 1번 링크 클릭 시 이동하고 싶은 위치에 해당하는 태그에 `id="위치명"` 입력하기, 예) `<body id="top"></body>`
3. 1번 링크에 2번 아이디 연결하기 `<a href="#이동아이디명></a>` **(#필수)**
## 이미지(img) - inline
* `<img src ="이미지경로" alt="대체텍스트">`
`<img src="./naver_logo.png" alt="로고">` (x)
`<img src="./naver_logo.png" alt="네이버">` (o)
`<img src="./newneek_logo.png" alt="뉴닉">` (o)
`<img src="./cart.png" alt="장바구니 버튼>` (x)
`<img src="./user.png" alt=로그인하기">`(o)
`<img src="./user.png" alt=로그인">`(o)
`<img src="./user.png" alt=로그인버튼">`(x)
`<img src="./user.png" alt=로그인아이콘">`(x)
### 이미지 필수 속성
* `src` : 이미지경로(**상대**/절대경로)
* `alt` : 대체텍스트(의미가 있는 경우 `alt="의미"`)
### 이미지 선택 속성
* `class`, `id` (이미지 외 모든 태그 동일)
## background-image (css 배경이미지)
### img태그가 아닌  background-image 처리하는 경우
* 형제, 자식, 자손 요소와 겹쳐저서 디자인 된 경우
* 쇼핑몰 상품 썸네일, 히어로 배너, 광고배너 등..
### 태그에 배경이미지 적용하는 작성순서
* `<태그 style="">`
* `<태그 style="background-image:;">`
* `<태그 style="background-image:url(경로);">`
